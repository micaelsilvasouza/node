Baixar node.js

O site oficial do Node.js tem instruções de instalação para Node.js: https://nodejs.org

Depois de baixar e instalar Node.js em seu computador, vamos tentar exibir "Hello World" em um navegador da web.
Crie um arquivo Node.js chamado "teste.js" e adicione o seguinte código: 

    
        var http = require('http');

        http.createServer(function (req, res) {
            res.writeHead(200, {'Content-Type': 'text/html'});
            res.end('Hello World!');
        }).listen(8080);

Salve o arquivo em seu computador.
O código diz ao computador para escrever "Hello World!" se alguém (por exemplo, um navegador da web) tentar para acessar seu computador na porta 8080.

Node.js arquivos devem ser iniciados no programa "Command Line Interface" do seu computador.
Navegue até a pasta que contém o arquivo "teste.js"

O arquivo que você acabou de criar deve ser iniciado por Node.js antes de qualquer ação pode ocorrer.
Inicie sua interface de linha de comando, escreva node teste.js e pressione Enter

Agora, seu computador funciona como um servidor!
Se alguém tentar acessar seu computador na porta 8080, receberá um erro "Olá Mundo!" mensagem em troca!
Inicie seu navegador de internet e digite o endereço: http://localhost:8080

Para parar a execução ultilize o atalho CTRL + C
