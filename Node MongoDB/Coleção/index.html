Coleção

Uma coleção no MongoDB é o mesmo que uma tabela no MySQL

        const TesteSchema = mongoose.Schema({ //Criar schema da coleção
            titte: {
                type: String, //determina o tipo do campo
                require: true //determina a obrigatoriedade
            },
        
            data: {
                type: Date
            }
        })
        
        mongoose.model("testes", TesteSchema) // Criar a coleção

No MongoDB, uma coleção não é criada até que Obtém conteúdo. O MongoDB aguarda até que você insira um documento antes de realmente criar a coleção.
