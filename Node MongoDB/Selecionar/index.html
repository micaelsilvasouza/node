Selecionar

Para selecionar os dados use o metodo find no objeto de model do moongose:

        TesteModel.find().then(dados => {
            console.log(dados)
        }).catch(err => {
            console.log("Erro: " + err)
        })

Ele pode receber os mesmos parâmetros, que é usado para o MongoDB, possuindo a mesma sintaxe.

Além dos metodos then e catch, pode ser usados antes deles, o metodo sort para ordenar os dados, que recebe um objeto, onde informa o atributo que é usado para ordernar, e recebo o valor crecente ou decrecente

TesteModel.find().sort({value: "desc "}).then(dados => { console.log(dados) }).catch(err => { console.log("Erro: " + err) })
