Atualizar

Para selecionar os dados use o metodo updateOne, funcionando igual ao do MongoDB:

        TesteModel.updateOne({_id: "675f46a60ed7725efee67e38"}, {title: "Atualizado", value: 456})
        .then(() => {
            console.log("Atualizado com sucesso")
        }).catch(err => {
            console.log("Erro: " + err)
        })

Ainda possui o metodo updateMany, que Ã© usado para o MongoDB, possuindo a mesma sintaxe.

        TesteModel.updateMany({}, {title: "Atualizado", value: 456})
        .then(() => {
            console.log("Atualizado com sucesso")
        }).catch(err => {
            console.log("Erro: " + err)
        })

E o metodo que atualizar e retorna os dados::

        TesteModel.findOneAndUpdate({_id: "675f46a60ed7725efee67e3a"}, {title: "Novamete de volta", value: 665})
        .then((dados) => {
            console.log("Atualizado com sucesso")
            console.log(dados)
        }).catch(err => {
            console.log("Erro: " + err)
        })
