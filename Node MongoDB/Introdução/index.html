Introdução

Node.js pode ser usado em aplicativos de banco de dados. Um dos bancos de dados NoSQL mais populares é o MongoDB.

Para baixar e instalar o driver mongoose, abra o Terminal de Comando e execute o seguinte:

        npm install mongoose

Node.js pode usar este módulo para manipular bancos de dados MongoDB:

        const mongoose = require("mongoose");

Criar o banco de dados

        const mongo = require("mongodb").MongoClient

        mongo.connect("mongodb://127.0.0.1:27017/blog", (err, db)=>{
            if( err )throw err
            console.log("db criado")
            db.close()
        })

O MongoDB só criar realmente um banco de dados, quando é adicionada algum dado.
