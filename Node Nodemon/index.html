Node Nodemon

Modulo node que automatiza a reiniciação do servidor node, a cade vez que o arquivo for alterado.

Use o comando para instalar:

    
        npm install nodemon -g

O -g funciona para instalar de forma global para funciona em qualquer outro projeto.

O modulo agora funciona subtituindo o termo node, que é usado quando você executa um arquivo javascript no computador. Para rodar o servidor digite o comando:

        nodemon servidor.js

Agora qualquer alteração no arquivo servidor.js, fará o node reiniciar o servidor automaticamente.
