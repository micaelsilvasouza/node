Modulo HTTP

Node.js possui um módulo integrado chamado HTTP, que permite que Node.js transfiram dados sobre o Hyper Text Transfer Protocol (HTTP). 

        var http = require('http');

O módulo HTTP pode criar um servidor HTTP que escuta portas do servidor e dá uma resposta de volta para o cliente. 

    
        http.createServer(function (req, res) {
            res.writeHead(200, {'Content-Type': 'text/html'});
            res.write(req.url);
            res.end();
        }).listen(8080);
  • res.writeHeade(): Adicionar um cabeçalho HTTP - primeiro argumento do método é o código de status, 200 significa Isso tudo está OK, o segundo argumento é um objeto que contém os cabeçalhos de resposta
  • res.write(): Escreve a mensagem que será retornada ao final da solicitação
  • req: Objeto que contem as informações da solicitação - req.url, retorna a url da solicitação que vem depois do dominio
  • res.end(): Finaliza a resposta que será encaminhada
