Rotas externas

Com o Express, é possivel criar rotas em arquivos separadados, para serem usados no arquivo principal

Crie um arquivo chamado rota-teste.js, adicione o Express, e use o sequinte código para criar uma rota:

        const rota = express.Router()

        rota.get("/", (req, res)=>{
            res.send("página principal da rota")
        })

Onde usando o Express, criamos uma constante, que recebe uma intancia de Router, e funciona com se fosse uma criação de uma rota normal

Para fazer com que seja usada na página principal, defina:

        app.use("/rotateste", rota)

Onde todas as rotas criadas no arquivo rota-teste.js, pode ser acessada, a partir da rota principal, pela rota /rotateste
