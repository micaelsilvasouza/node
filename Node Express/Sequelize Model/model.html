Sequelize Model

Model é uma forma do sequelize criar tabelas, sem a necessidade de digitar comandos SQL

Para criar um model usamos a função define(), que cria uma tabela.

        const tabela = sequelize.define(nome, campos)
  • nome - Nome da tabela
  • campos - Campos de informações da tabela, informados no formato object
        const Usuario = sequelize.define("usuarios", {
            nome: {
                type: Sequelize.STRING
            },
            sobrenome: {
                type: Sequelize.STRING
            },
            email: {
                type: Sequelize.STRING
            },
            dataNascimento: {
                type: Sequelize.DATEONLY
            }
        })
    
        Usuario.sync({force: true})

Usuario.sync({force: true}), cria a tabela, e o argumento funciona para força a criação da tabela, garantindo que seja sempre criada.

Para inserir dados na tabela, use a função create().

        Usuario.create({
            nome: "André",
            sobrenome: "Silva",
            email: "andresilva@gmail.com",
            dataNascimento: "1990-02-15"
        })

Para acessar todos os dados use a função all:

        Usuario.all().then((dados)=>{
            console.log(dados)    
        })
