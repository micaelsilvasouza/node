Criar variáveis globais

Variáveis globais podem ser usadas em qualquer parte do handlebars da aplicação, e para criar uma dentro de uma função express, de requerimento e resposta, basta usar res.locals

        app.use((req, res, next)=>{
            res.locals.nome_da_variavel = "um valor javascript"
        })
        
        app.get("/", (req, res)=>{
            res.locals.nome_da_variavel = "um valor javascript"
        })
