Sequelize FindAll

FindAll Ã© usado para buscar dados em uma tabela

Busca todos os dados no model Registro:

        Registro.findAll().then((registros)=>{
            console.log(registros)
        })

Busca apenas dados especificos no model Registro:

        Registro.findAll({attributes: ["id", "horario", "descricao"]}).then((registros)=>{
            console.log(registros)
        })

Buscar dados usando a WHERE, encotrando os dados para quando o id for igual a 1:

        Registro.findAll({where: {id: 1}}).then((registros)=>{
            console.log(registros)
        })

Para mais filtros adicione mais atributos ao objeto "where"
