Parâmetros

Parâmetros são dados que podem ser passados na url, mas funcionam diferentes de dados enviados por requisições do tipo get.

Funcionam mais precisamente como um caminho em uma url, exemplo

        localhots:8081/pagina/dadoA/dadoB
  • localhots:8081 - O host onde a pagina se encontra, junto com a porta do servidor
  • pagina - A rota da pagina que pede os parâmetros
  • dadoA - O dado na forma de parâmetro passado para a url
  • dadoB - O dado na forma de parâmetro passado para a url

Para criar a opção de parâmetros na pagina é necessário informa-los no primeiro argumento do método get, onde criamos a rota, usando a sintaxe /:param_name.
Podemos criar mais de um, simplemente adicionando um após o outro /:paramA/:paramB.
