Sequilize

Um modulo para trabalhar com bancos de dados dentro do node

Use o comando para instalar:

        npm install --save sequelize

Para trabalhar com bancos de dados mysql é preciso instalar o modulo para trabalhar com mysql

Use o comando para instalar o modulo mysql2:

        npm install --save mysql2

Lembrando que todos esses comandos instalam os modulos, apenas no local do projeto, para instalar globalmente, use -g no lugar --save

Para criar uma conexão ao banco de dados importe o modulo, que representa algo que usaremos para criar uma conexão

    
        const Sequelize = require("sequelize")
        const sequelize = new Sequelize(db, user, pass, {host, dialect})
  • Sequelize - Módulo importado
  • sequelize - Objeto de conexão criado por Sequelize
  • db - Database name, nome do banco de dados que deseja conectar
  • user - Nome de usuário que usamos para acessar o mysql
  • pass - Senha de usuário para a conexão
  • {} - Objeto contendo propriedades para a conexão
    • host - Host de conexão do banco de dados
    • dialect - O tipo de banco de dados a ser usado

Use a função authenticate para verificar se a conexão foi bem sucedida

    
        sequelize.authenticate().then(()=>{
            console.log("Conectado com sucesso!")
        }).catch((err)=>{
            console.log("Falha ao se conectar!")
        })
