Express POST

Express pode trabalhar com requisições do tipo post, para isso use a função post para criar uma rota do tipo post

        app.post("/post", (req, res)=>{
            res.send("Formulario recebido")
        })

Para utilizar os dados recebidos por post, precisamos do módulo body-parse, use o comando para instalar:

        npm install body-parser

Para configura a utilização do módulo:

        app.use(bodyParser.urlencoded({extended: false}))
        app.use(bodyParser.json())

E use o argumento de requisição para acessar o dado recebido

        req.body.nome
