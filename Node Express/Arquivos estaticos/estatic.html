Usando arquivos estáticos

Os arquivos estáticos, sãos os arquivos css e js, que fazem parte do front-end.

Para configurar os usos de arquivos estáticos:

        app.use(express.static(__dirname + "/estaticos"))

Dentros de express.static, é passado como argumento o diretório da pasta, onde ficam os arquivos estáticos.

A variável __dirname, é usada para retornar o diretório local, assim evitar erros.

Caso a pasta fique em um diretório diferente, basta usar o modulo path

        app.use(express.static(path.join(__dirname + "/estaticos")))

O metodo uni os diretórios

Uma boa ideia é usar o bootstrap como diretório, pois facilita na criação do front-end
