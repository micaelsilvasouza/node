Middleware

Middleware, também conhecido como “software intermediário”, é uma camada de software que age como uma ponte entre sistemas operacionais e aplicações em uma rede distribuída. A principal função do middleware é facilitar a comunicação e a troca de dados entre aplicações de software diferentes

Para adicionar no Express:

        app.use((req, res, next)=>{
            //code
                ////
            next()
        })

Possui uma sintaxe semelhante à uma rota, mas possui o parametro next, que é usado para passar para a proxima etapa.
