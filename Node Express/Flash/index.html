Flahs

Instale o connect-flash, para auxliar nas comunicações entre meddleware e rotas, ou de rotas para rotas:

        npm install connect-flash

Ou instale express-flash, que funciona semelhante e é mais atual:

        npm install express-flash

Para usar o flash, é preciso usar uma sessão

Para configurar o flash, adicione sempre abaixo da configuração de sessão:

        //configurando flash
        app.use(flash())

Para defini um valor de flash em uma rota, ou meddleware, faça dentro da função de rota ou middleware:

        req.flash("indentificador", "valor")

E para usar o valor faça:

        req.flash("indentificador")

O valor retornado é uma matriz, que contem todo os valores que foram adicionados ao "indentificador" flash
