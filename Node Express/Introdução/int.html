Node Express

Express é um framework, que criar um servidor web usando node

Use o comando para instalalo

        npm install express

Importe o express usando o comando: 

        const express = require("express")

Para criar um página para a rota principal, use o cógio a seguir: 

    
        const express = require("express")
        const app = express()

        app.get("/", (req, res)=>{
            res.send("Essa é minha página criada usando Express no Node!")
        })

        app.listen(8081, ()=>{
            console.log("Servidor rodando na url http://localhost:8081")
        })
  • app - Recebe todas as funções de Express e é responsavél por fazer o servidor gerar as páginas
  • get - Cria uma rota dentro da aplicação
  • listem - Excuta o servidor na porta especificada
