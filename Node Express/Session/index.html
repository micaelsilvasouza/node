Session

Instale o móldulo para trabalhar com sessões no Express:

        npm install express-session

Instale o connect-flash, para auxliar nas sessões:

        npm install connect-flash

Para configurar a sessão faça:

        const session = require("express-session")
        const flash = require("connect-flash")
        
        // Configurando sessão
        app.use(session({
            secret: "umvalor",  // valor chave da sessão, podendo ser qualquer valor
            resave: true, 
            saveUninitialized: true
        }))

Onde o atributo secret é valor chave da sessão, podendo ser qualquer valor, sendo ideal ser um valor bem seguro

Para configurar o flash, adicione sempre abaixo da configuração de sessão:

        //configurando flash
        app.use(flash())
