Handlesbars

Para exibir partes padrões, voce deve criar dentro da pasta views, uma pasta chamada partials.

Dentro da pastas crie um arquivo Handlesbars, pode usar qualquer nome, como exemplo padrao.handlebars.

Nesse arquivo crie a estrutura html que deseja.

Para inserir nas página, o arquivo dentro da página layout, que está sendo usado, adicione na parte deseja:

        {{>nomeDoArquivo}}
