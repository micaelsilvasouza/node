Handlesbars

Handlebars serve para exibir informações do back-end de uma forma robusta no HTML.

Use o comando para instalar:

        npm install express-handlebars

Para fazer com que o Node Express use o Handlebars, use as linhas de configuração:

        app.engine("handlebars", handlebars.engine({defaultLayout: "main"}))
        app.set("view engine", "handlebars")

Agora dentro da pasta onde se encontra o projeto, ou seja o arquivo js que contem o app Express.
Crie o diretório views/layouts
Dentro desse diretório crie um arquivo chamado main.handlebars
Dentro desse arquivo ficará o HTML que servirá de padrão a ser usado pelo handlebars na construção do projeto.

Dentro do body do arquivo handlebars coloque:

        {{{body}}}

Isso chamará um arquivo, com a extenção handlebars dentro da pasta views, que contem o conteudo HTML e adicionara dentro da chamada:

        app.get("/",(req, res)=>{
            res.render(name, obj)
        })
  • name - nome do arquivo, sem necessidade da extenção handlebars
  • obj - Objeto contendo propriedades são exibidas dentro do arquivo, usando pares de chaves ({{prop}})
