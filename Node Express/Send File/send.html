Send File

A função sendFile() envia um arquivo para que acessar o servidor.
Como argumento ela recebe o diretório de onde o arquivo está localizado.
Para evitar erro quando o arquivo se encontra no mesmo diretório, use a variavel __dirname, para retorna o diretório completo de onde seu arquivo js se encontra, basta concatenar com o resto do diretório. 

    
        app.get("/", (req, res)=>{
            res.sendFile(__dirname + "/send.html")
        })

Assim essa página pode ser enviada e visualizada quando a pessa acessa o servidor
